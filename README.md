🌍 GitHub Pages Hosting

These projects are hosted using GitHub Pages free static hosting.

GitHub Pages allows hosting static websites directly from a GitHub repository.

To deploy:

1. Go to Repository Settings
2. Open Pages
3. Select branch (main)
4. Save
5. Website will be live

## Viewing Locally

To explore the portfolio on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/kriss2012/Portfolio.git
    ```
2.  **Navigate into the project directory:**
    ```bash
    cd Portfolio
    ```
3.  **Open the `Portfolio.html` file** in your preferred web browser.
    *   You can typically do this by double-clicking the file in your file explorer, or by using a command like `start Portfolio.html` (Windows) or `open Portfolio.html` (macOS) in your terminal.
